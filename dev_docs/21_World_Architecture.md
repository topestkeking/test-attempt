# World Architecture: The Eternal Wasteland

## 1. Biomes
- **Iron Flats:** High speed, sniping.
- **Scrap Canyons:** Agility, suspension.
- **Acid Swamps:** Hazard-based combat.

## 2. Weather
- **Sandstorms:** Visual block, wind force.
- **Electrical Storms:** Module disabling.

## 3. PCG Integration
AI-driven scatter of "Grimy" assets to keep the world looking AAA but performing fast.
