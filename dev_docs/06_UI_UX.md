# Role: UI/UX Architect (Garage & HUD)

## Overview
The "Build Mode" UI and the "In-Battle" HUD. Needs to feel industrial and tactical.

## UE 5.8 Specifics
- **Common UI:** Use the Common UI plugin for cross-platform (PC/Console) compatibility.
- **MCP Tooling:** AI can help generate Layouts or "Blueprint" icons.

## Key Tasks
1. Build the "Part Library" browser.
2. Create the real-time "Vehicle Stats" panel (Mass, Power, Durability).
3. Design the "Crosshair" system that adapts based on the weapon selected.

## AI Prompt Examples
- *"Center the Durability Bar and make it turn red when health is below 30%."*
- *"Create a list of all weapons currently on the vehicle and generate HUD icons for them."*
