# Role: Physics Configuration (The Unreal Native Way)

## Overview
Your job is to tweak the settings in the **Chaos Vehicle Movement Component** to make the "Cars with Guns" feel awesome. No custom code required.

## Key Tasks
1. Adjust the `Suspension` stiffness so the car doesn't flip when heavy gun turrets fire.
2. Configure the `Tire Config` for different wasteland surfaces (Sand, Metal, Salt).
3. Use the `Center of Mass Offset` to keep the car grounded during high-speed turns.

## AI Prompt Examples
- *"Claude, look at the Chaos Vehicle component. Increase the torque curve by 15% and set the suspension damping to be slightly stiffer for a heavier build."*
- *"Adjust the friction coefficient on the 'Sand' Physical Material to allow for more drifting."*
