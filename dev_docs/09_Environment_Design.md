# Role: Environment Lead (The Wasteland)

## Overview
Designing the arenas where the battles happen. High-performance, grimy, and vertical.

## UE 5.8 Specifics
- **PCG (Procedural Content Generation):** Use 5.8 PCG to scatter scrap piles, rocks, and debris.
- **World Partition:** Enable seamless loading for larger "Adventure Mode" maps.

## Key Tasks
1. Create "Battle Arenas" with balanced cover and sniping lanes.
2. Setup the "Toxic Sandstorm" or "Acid Rain" environmental hazards.
3. Optimize collision for the landscape to ensure vehicles don't "clip" through.

- *"Use PCG to scatter 50 rusted barrel actors across this landscape section."*
- *"Adjust the landscape heightmap to create a natural ramp for vehicles."*
