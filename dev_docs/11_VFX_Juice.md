# Role: VFX Artist (Fire & Smoke)

## Overview
Explosions, exhaust fumes, and sparks. The "juice" of the combat.

## UE 5.8 Specifics
- **Niagara:** Utilize Niagara Fluine for high-fidelity fire and smoke.
- **VDB Support:** Use 5.8's improved VDB rendering for cinematic explosions.

## Key Tasks
1. Create a "Black Smoke" exhaust trail for damaged engines.
2. Setup the "Blue Flash" for energy weapons or "Orange Flare" for ballistics.
3. Implement vertex-offset "Impact Wobble" for the vehicle chassis.

## AI Prompt Examples
- *"Increase the particle count of this Niagara explosion by 50% and add a secondary fire trail."*
- *"Spawn a spark effect at the exact location where a projectile hits a metal block."*
