# Role: Localization & Global Accessibility

## Overview
Making the game playable in every language and for players with disabilities.

## UE 5.8 Specifics
- **AI Translation:** Use the AI to auto-generate translation keys for English, Spanish, Russian, etc.
- **Colorblind Modes:** Implement 5.8's accessibility post-process filters.

## Key Tasks
1. Setup the "LocText" tables for all UI.
2. Implement "Subtitles" for all voice-over.
3. Design "One-Handed" control schemes for accessibility.

## AI Prompt Examples
- *"Translate the current UI text to Russian and ensure it fits within the button boundaries."*
- *"Enable a Protanopia colorblind filter on the main camera."*
