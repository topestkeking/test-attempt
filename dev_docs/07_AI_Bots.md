# Role: AI & Bot Specialist

## Overview
Programming the NPC enemies that challenge the player in PvE or fill empty PvP slots.

## UE 5.8 Specifics
- **Mass Entity:** Use the 5.8 Mass Framework for large-scale bot battles.
- **NavMesh:** Ensure bots can navigate modular terrain and avoid falling into traps.

## Key Tasks
1. Write logic for bots to "Aim" at player Cabins.
2. Develop "Squad Tactics" (flanking, retreating).
3. Create a "Build Generator" that randomly constructs valid bot vehicles from the part list.

## AI Prompt Examples
- *"Create a Behavior Tree task that makes the bot maintain a 100-meter distance from the player."*
- *"Randomly select 5 armor blocks and 1 weapon to build a level 1 bot vehicle."*
