# Role: Economy & Progression Balance

## Overview
Designing the "Grind." How players unlock new scrap, weapons, and prestige.

## UE 5.8 Specifics
- **Data Tables:** Extensive use of Data Tables (accessible via MCP) to balance part stats.
- **Save Games:** Modern UE5 save systems for complex vehicle blueprints.

## Key Tasks
1. Define "Scrap Metal" as the primary currency.
2. Setup the "XP" curve for player levels.
3. Balance "Power Score" (PS) to ensure players only fight similarly powerful vehicles.

## AI Prompt Examples
- *"Calculate the average time to unlock a 'Purple' rarity weapon based on 5 matches per day."*
- *"Add a 5% health buff to all 'Heavy' cabin types in the Data Table."*
