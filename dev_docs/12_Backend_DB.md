# Role: Backend & Database Engineer

## Overview
Managing player accounts, vehicle blueprint storage, and match-making.

## UE 5.8 Specifics
- **Epic Online Services (EOS):** Integration for cross-play and friends lists.
- **MCP for Server Logic:** AI-assisted scripting for backend microservices.

## Key Tasks
1. Setup the "Blueprint Gallery" (where players can upload/download car designs).
2. Implement the Matchmaking queue logic.
3. Secure the database against "illegal" part counts in saved vehicles.

## AI Prompt Examples
- *"Write a JSON schema for storing a vehicle blueprint, including part IDs, rotations, and scales."*
- *"Query the EOS backend to find the current player count in the 'Scrap Metal' queue."*
