# Role: Quality Assurance & Automation

## Overview
Breaking the game so players don't. Testing physics edge cases.

## UE 5.8 Specifics
- **Gauntlet Automation:** Use UE 5.8 Gauntlet to run stress tests on vehicle physics.
- **MCP Auto-Test:** AI can "Spawn 100 random cars" and report crashes.

## Key Tasks
1. Create a "Test Map" with ramps, hammers, and cliffs.
2. Automate the "Collision Test" (ensure parts don't clip when resized).
3. Log frame rates during 8-player physics battles.

## AI Prompt Examples
- *"Run a 10-minute automated battle between two bots and log any physics 'explosions' or glitches."*
- *"Verify that the vehicle doesn't fall through the floor when scaled to 5x size."*
