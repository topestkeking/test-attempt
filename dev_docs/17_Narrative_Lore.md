# Role: Narrative & Lore Specialist

## Overview
Why are we building these death machines? The story behind the wasteland.

## UE 5.8 Specifics
- **Text-to-Speech / AI Voice:** Use modern plugins to generate "Scavenger" dialogue.
- **Lore UI:** Design a "Wasteland Encyclopedia" using Common UI.

## Key Tasks
1. Write the history of the "Great Rusting."
2. Create names and descriptions for every part (e.g., "The Widowmaker Cabin").
3. Design the "Factions" (e.g., The Scrap-Kings vs. The Chrome-Legion).

## AI Prompt Examples
- *"Write a 3-sentence description for a rusted V8 engine found in a scrapyard."*
- *"Generate a list of 10 scavenger-themed names for a desert faction."*
