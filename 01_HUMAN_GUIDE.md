# 🛠️ 01_HUMAN_GUIDE.md: Onboarding & Handbook

This is the primary guide for any human joining the "Cars & Cannons" team. It summarizes the project vision and your role in the AI-human collaboration loop.

## 1. The Vision: "Cars with Guns"
- **Gameplay**: Modular vehicle builder meeting competitive PvP.
- **Aesthetic**: "Mad Max" grimy, rusted, industrial.
- **Goal**: Fun > Reality. If it feels cool, build it.

## 2. Your Workflow
We use **Claude Code** and the **Unreal MCP Plugin** as a force multiplier.
- **The Loop**: You architect the data and vision -> Claude implements the boilerplate and technical math -> You verify and "Juice" the result in the editor.
- **The Checklist**:
  1. Complete the technical setup in `TECHNICAL_INSTALLATION_CHECKLIST.md`.
  2. Read the governance rules in `PROJECT_MEMORY/00_AGENT_RULES.md`.
  3. Start with the `L_Garage_Hub` level to understand the snapping logic.

## 3. The 3 Action Streams
- **Stream A (The Heart)**: Physics, Building, Combat.
- **Stream B (The Frontier)**: Art, Environments, VFX.
- **Stream C (The Command Center)**: UI, Backend, Economy.

## 4. Communication
- Everything we do is recorded in `PROJECT_MEMORY/`.
- If you make a major design change, update `PROJECT_MEMORY/03_DECISION_LOG.md`.
