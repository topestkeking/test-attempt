# 🛠️ TEAM ONBOARDING: Project "Cars & Cannons"

Welcome to the team! This sheet will get you up to speed in 5 minutes. We are building a "AAA+++" grimy vehicle-builder PvP game in Unreal Engine 5.8.

---

## 1. The Vision: "Crossout meets Mad Max"
- **Gameplay:** Build a custom car from scrap blocks, put big turrets on it, and fight other players.
- **Vibe:** Grimy, rusted, industrial, and high-octane.
- **Goal:** Fun and competitive. If it feels cool to drive and shoot, we're winning.

## 2. Your Development Partner: The AI (Claude)
We use **Claude Code** and the **Unreal MCP Plugin**. Think of the AI as a 3rd team member who can:
- Write complex math (when we need it).
- Set up boilerplate Blueprint logic.
- Run tests in the engine.
- *Tip:* Check out the `MCP_AI_Setup_Guide.md` to get your local environment connected.

## 3. Where to Start? (The "Dev Docs" Folder)
The project is broken down into 20+ specialized categories in the `dev_docs/` folder.
- **Physics/Driving:** Read `Physics_Deep_Dive.md`.
- **Art/Aesthetic:** Read `dev_docs/04_Technical_Art.md`.
- **Old-School to Modern:** If you're used to older coding styles, read `dev_docs/24_Legacy_Rosetta_Stone.md`.

## 4. The Development Flow
1. **The Garage:** This is our safe zone. We focus on the "Build" UI and part snapping here.
2. **The Gauntlet:** Our testing arena. If you make a new gun or part, test it here first.
3. **The Wasteland:** Our battle arenas. This is where the PvP happens.

## 5. Key Rules for the Team
- **Fun > Reality:** If a physics law makes the game boring, break it.
- **Stay Grimy:** Everything should look like it was welded together in a junkyard.
- **Use the AI:** Don't waste time on repetitive tasks. Ask Claude to help you generate assets, code, or documentation.
