# 🤝 04_AGENT_HANDOFF_FORMAT.md: Inter-Agent Communication

Use this format when passing a task from one agent/specialty to another.

## Handoff Summary
- **Source Agent**: (e.g., Physics Agent)
- **Target Agent**: (e.g., QA Agent)
- **Status**: (Completed / Blocked / Partial)

## Changes Made
- List specific logic or assets modified.

## Tests Run
- Verification results (Gauntlet logs, etc.).

## Remaining Work
- What the next agent needs to finish.

## Context for Next Agent
- "The suspension is tuned, but the center of mass might be too high for the Heavy class. Please verify in the Gauntlet."
